# 雙動能交易策略回測報告

此報告詳細說明了雙動能策略的執行過程與結果。

## 1. 資料處理
- 使用 `資料2.xlsx` 中的 16 種商品周收盤價。
- 處理缺失值時採用先 `ffill()` 後 `bfill()`，確保不包含未來資訊。

## 2. 策略邏輯
- 每周計算動能，選取動能為正且排名前 2 的商品。
- 若商品續留，則保持股數不變；若為新商品，則分配剩餘資金或重新分配（依據 10M 初始資金比例）。

## 3. 最佳化結果
- 採用螞蟻演算法 (ACO) 進行全域與逐標的最佳化比較。
- 最終選用 Calmar Ratio 較高的方法。
